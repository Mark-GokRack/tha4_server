# tha4_server


## Instruction

### for windows
    ```cmd
    python -m venv .venv --prompt tha4_srv
    .venv\Scripts\activate.bat
    python.exe -m pip install --upgrade pip
    pip install -r requrements.txt
    ```

### for linux
    ```cmd
    python -m venv .venv --prompt tha4_srv
    source .venv/bin/activate
    python -m pip install --upgrade pip
    pip install -r requrements.txt
    ```
